Decklight

A deck is a single HTML file, authored by your favorite Agent preferably, or yourself.
No build step, no server, no framework.

Decklight is built in the Reveal.js tradition, with every feature designed so a machine can prove the deck works — WCAG-gated themes, truthful terminal recordings, and a voice that can present for you. The deck below presents itself.

One file, zero build

Author a single HTML file, double-click it, present. file:// is enough — and decklight bundle flattens a whole course into one email-able file.

Agent-native authoring

Describe a slide in plain English and an agent answers in Decklight — with overflow guards, contrast gates and headless render checks standing in for eyeballs.

Keynote-grade motion

Progressive builds from one attribute, Magic Move between slides, and theme-aware SVG diagrams that literally draw themselves in.

61 themes, one contract

Every theme passes WCAG gates and codified palette rules — diagrams and terminals reskin too. Press ⌃T in the deck and it generates a brand-new one.

Terminals & a voice

Real PTY recordings replayed truthfully — never video — plus live TTS narration that presents the deck by itself, captions and pop quiz included.

Get started in thirty seconds

No install — npx runs the CLI straight from npm. This terminal is Decklight's own player, replaying a recorded session (press ▶ to run it again).
